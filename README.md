# Compass-plugin
Plugin for Paper 1.19.3
Плагин для ядра Paper 1.19.3

В плагине реализован кастомный крафт на компас, показывающий корды игрока, после чего ломается.

Конфиг
``` java
special_compass:
  display_name: '&aCompass'
  recipe:
    - '111'
    - '121'
    - '111'
  ingredients:
    2: 'DIAMOND'
    1: 'IRON_INGOT'
```
